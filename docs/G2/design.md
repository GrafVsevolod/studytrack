## Прототип (клики)
- Login → Dashboard
- Dashboard → Create Goal / Create Task / Statistics
- Create Goal → Dashboard (Save/Cancel)
- Create Task → Dashboard (Save/Cancel)
- Statistics → Dashboard

## Примечания
Макеты сделаны в стиле, близком к Material UI (MUI), чтобы реализация совпадала с дизайном.

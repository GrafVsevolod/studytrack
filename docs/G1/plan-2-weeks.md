# План работ на 2 недели (G1)

Цель — подготовить все артефакты для контрольной точки G1: описание проекта, макеты, схема данных и план.

## Неделя 1

### Задача 1. Описание проекта и целей

- **Описание:**  
  Сформулировать проблему, решение, целевую аудиторию и основные функции StudyTrack.
- **Результат:**  
  Заполненный файл `docs/G1/presentation.md` (разделы 1–8).
- **DoD (Definition of Done):**
  - Описана проблема и зачем нужно приложение.
  - Описано решение и кто целевая аудитория.
  - Перечислены основные функции (MVP).
  - Указан планируемый технологический стек.

---

### Задача 2. Пользовательский сценарий (flow)

- **Описание:**  
  Подробно расписать шаги пользователя от первого входа до просмотра статистики.
- **Результат:**  
  Разделы 5 и 6 в `presentation.md` + краткое описание в `wireframes.md`.
- **DoD:**
  - Описан путь нового пользователя (первый вход).
  - Описан путь возвращающегося пользователя.
  - Понятно, какие экраны участвуют и как они связаны.

---

### Задача 3. Архитектура и поток данных

- **Описание:**  
  Определить основные модули фронтенда и бэкенда, способы их взаимодействия, использование Web Worker и Canvas API.
- **Результат:**  
  Заполненный `docs/G1/dataflow.md` + схема на доске.
- **DoD:**
  - Описаны основные блоки (UI, Redux, Backend, DB, Web Worker).
  - Описано, какие API и протоколы используются.
  - Указано, как подключаются аналитика и баг-трекер.

---

## Неделя 2

### Задача 4. Макеты основных экранов (wireframes)

- **Описание:**  
  Нарисовать простые схемы (без дизайна) для 5 экранов: логин, дашборд, создание цели, создание задачи, статистика.
- **Результат:**  
  Доска в Miro + ссылка в `docs/G1/wireframes.md`.
- **DoD:**
  - Есть макеты всех пяти экранов.
  - По макетам можно пройти весь пользовательский сценарий.
  - На макетах подписаны основные элементы интерфейса.

---

### Задача 5. Репозиторий и структура G1

- **Описание:**

- **Результат:**  
  Публичный репозиторий с папкой `docs/G1` и заполненными файлами.
- **DoD:**
  - Репозиторий доступен по ссылке.

- **Результат:**
